# Python Geometry Visualizer

A Python project for modeling basic geometric shapes, calculating measurements, and preparing visual outputs for learning and technical demonstration.

## Overview

This repository is being rebuilt from an earlier geometry assignment into a cleaner portfolio project focused on Python fundamentals, object-oriented design, and readable technical documentation.

## Current Goals

- Represent common geometric shapes in Python
- Calculate area and perimeter values
- Add simple validation for dimensions
- Build toward optional plotting or visual output
- Include tests for calculation accuracy

## Tech Stack

- Python
- Object-oriented programming
- Unit testing
- Geometry/math fundamentals

## Planned Features

- Shape classes for circles, rectangles, and triangles
- Area and perimeter calculations
- Input validation
- Unit tests
- Optional matplotlib visualizations

## Portfolio Purpose

This project demonstrates clean Python structure, class design, testing, and clear documentation around a small but understandable technical problem.
